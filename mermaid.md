# Mermaid

Gitbook 插件：支持渲染Mermaid图表

## Mermaid-gb3
```
npm install gitbook-plugin-mermaid-gb3
https://github.com/chriswessels/gitbook-plugin-mermaid-gb3

book.json
{
"plugins": [
    "mermaid-gb3"
]
}
```

## 展示效果
